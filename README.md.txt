MASTERMIND


RULES

1. The player has to guess a pattern developed by the coder which is of length 4 composed of the digits 0-5.

2. They decide on the number of rounds and the player enters his first guess,
	The coder responds to the player with a 4 digit code representing

	1- if the position and digit is correct
	0- if the position and digit is wrong
	2- if the digit exists but not in the position
HOW TO PLAY?

1. There are three ways to start the game
	a. using the predefined sequence
	b. random sequence generated
	c. two player where one player chooses the sequence

2. Select the number of rounds!

3. Start guessing

4. Find the sequence before the rounds run out and become a WINNER!!!
